Efficiency-corrected RT-qPCR

How to correct RT-qPCR Cq data for amplification efficiency — a method demonstration

Synthetic demo. Every value here is randomly generated. This is a method illustration, not a real experiment or result.

Why correct for efficiency?

The 2^(−ΔΔCt) shortcut assumes every assay doubles its product each cycle (100% efficiency). When a target and reference assay differ in efficiency, an uncorrected ratio is biased. Efficiency correction rescales each Cq to a common 100%-efficiency basis (the Pfaffl principle) before quantities are compared.

Calculation

Efficiency (%)   = (-1 + 10^(-1 / slope)) * 100
E                = 1 + Efficiency(%) / 100
Ct(100%)         = Cq * log2(E)
Quantity         = 100 * 2^( mean(std100 Ct(100%)) - sample Ct(100%) )
Normalized ratio = Target quantity / Reference quantity
